# Riz au lait | Client Unity

## Démarrer
 - Pour compiler le projet, il vous faudra la version 6000.0.58f2 d'Unity 6. 
 - Avant d'ouvrir le projet, il vous faudra décompresser les deux fichiers .7z présents dans `Assets\Inference Engine\Speech Recognition\Models`.
 - La scène finale se trouve dans le fichier Interface v4_test.
